Overview

Devnet framework is free open-source, PHP software and cross-platform, with high-performance, for building console and modern web application, with rapid application development tools.

Server requirements

Befor installing DevNet Framework, make sure your server meets the following requirements :

  • PHP 7.4 or higher version
  • JSON PHP Extension
  • PDO PHP Extension
  • Composer Dependency Manager

You can get the latest version of PHP from php.net
also you can get Composer the PHP Dependency Manager from getcomposer.org
and you may need Git the distributed version control system from git-scm.com

Installing DevNet

devnet/installer is a metapackage that uses composer dependency manager, to install DevNet libraries globally through command line into your system, with a single runtime library for all the DevNet projects.

But first we need to config composer global minimum-stability to development, because DevNet is still in development
so in the terminal type the following command: 


composer global config minimum-stability dev

To Install DevNet framework globally, run the following command in the terminal: 


composer global rquire devnet/installer

The installer will install four main DevNet packages:

  • devnet/core a basic common library required by all DevNet components.
  • devnet/cli a command line Interpreter that has development tool, for creating and running DevNet prjects
  • devnet/web a collection of web components that helps to create a web project in modern way.
  • devnet/entity an object relational mapping system that helps to manipulate database, in object-oriented way.
If you don't want the full installation of DevNet framework, you can install onley the packages that fit your need, and choose whether to make the installation globally outside your project, or localy inside your project, and you can do the same with the third party libraries.

Uninstalling DevNet

To uninstall DevNet Framework globally, run the following command in the terminal: 


composer global remove devnet/installer

Using DevNet Command:

For the first start you can run the devnet command with the option --help in your terminal, to help you how to use devent command line tool, as shown below: 


devnet --help

Note: For linux users, do not forget to add into the System Environment Variables the following line:
export PATH="$PATH:$HOME/.config/composer/vendor/bin"
