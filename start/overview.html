Overview

Devnet framework is free open-source, PHP software and cross-platform, with high-performance, for building console and modern web application, with rapid application development tools.

Server requirements

Befor installing DevNet Framework, make sure your server meets the following requirements :

  • PHP 7.4 or higher version
  • JSON PHP Extension
  • PDO PHP Extension
  • Composer Dependency Manager

You can get the latest version of PHP from php.net
also you can get Composer the PHP Dependency Manager from getcomposer.org

Installing Devnet

To install Devnet Framework project please follow the following steps :

  1. Download or clone DevNet installer repository from here
  2. Rename the folder project any name you want
  3. In the terminal, navigate to the root folder of your project and run the following command :

composer install

This will install only DevNet SDK with Components and liberies needed for your project, and in order to initialize and manage your project, use devnet command line tool that is provided by DevNet SDK.

To know how to use devent command line tool use the options --help like in the following command: 


./devnet --help

Note: If you are using Linux do not forget to give execute permission to devnet command: chmod +x ./devnet
