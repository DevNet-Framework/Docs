Console Application

To create a Console Application, enter the following command in the Terminal : 


devnet new console

The template creates a simple "Hello World" application, has a class Program with a single static method Main, that calls the static method Console::writeLine(String $string) which writes data to the output stream, followed by a new line terminator. 


                  <?php

                  namespace Application;

                  use Artister\System\Console;

                  class Program
                  {
                     public static function main(array $args = [])
                     {
                        Console::writeLine("Hello World!");
                     }
                  }

Run the console application

To run the application, run the following command in the Terminal : 


devnet run

The program displays "Hello World!" and ends. 


devnet run
Hello World!

Command line arguments

At the runtime, the input arguments are passed to the parameter $args of the static method main(array $args) in form of indexed array, and to handle these arguments just handle the values of the parameter $args.

The following example, shows a simple code that handle the input arguments, it stores the value of the first argument of the array $args into the variable $name, using Null Coalescing operator, if the it is not set, it will set to the default value "world", then pass it to the static method write(string $string, ?array arguments) to write the content Hello {name} to terminal, using formatted string pattern. 


                  <?php

                  namespace Application;

                  use Artister\System\Console;

                  class Program
                  {
                     public static function main(array $args = [])
                     {
                        $name = $args[0] ?? "World";
                        Console::write("Hello %s!", $name);
                     }
                  }
Output Example

devnet run Job
Hello Job!

Note: The static method write does not insert a new line at the end of the text, use the escape sequence \n or use the static method writeLine instead, but keep in mind that the writeLine method does not support the formatted string like the write method.

Console input

The static method Console::readLine(?string $value) reads the next line of characters from the console input stream, and returns it as a string data, it will pause the program execution, until the user type a data and presses the Enter key, optionally can write as text befor reading the input user.

The following example code, displays a prompt in the terminal asking the user's name, and after the user input his name followed by the Enter key, it will return a hello message to the user by his name. 


                  <?php

                  namespace Application;

                  use Artister\System\Console;

                  class Program
                  {
                     public static function main(array $args = [])
                     {
                        Console::writeLine("What is your name?");
                        $name = Console::readLine();
                        Console::writeLine("Hello {$name}!");
                     }
                  }
Output Example

devnet run
What is your name?
Job
Hello Job!

Console Color

The Console class has the static methodes foregroundColor(int $colorCode) and backgroundColor(int $colorCode) to set the foreground and background color of the console output respectively, and the static method ResetColor() to restore the output system to the original console color.

The ConsoleColor class has a list of color constants that are supported by the system, each constant hold a color code value, this will help to set the color by name without need to remember the color code, like in the folowing example: 


                  <?php

                  namespace Application;

                  use Artister\System\Console;
                  use Artister\System\ConsoleColor;

                  class Program
                  {
                     public static function main(array $args = [])
                     {
                        Console::foregroundColor(ConsoleColor::Yellow);
                        Console::write("Hello");
                        Console::backgroundColor(ConsoleColor::Magenta);
                        Console::writeLine(" World!");
                        Console::resetColor();
                     }
                  }
Output Example

devnet run
Hello World!
