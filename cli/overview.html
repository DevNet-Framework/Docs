Overview

Devnet framework is free open-source software, and cross-platform, built in PHP with high-performance, for rapid application development, to build APIs, console and modern web application.

DevNet CLI

DevNet Command-Line Interface is a tool that used for creating and running DevNet projects, built on the top of DevNet System, and can be extensible via plugins and packages.

Server requirements

Befor installing DevNet CLI, make sure your server meets the following requirements :

  • PHP 7.4 or higher version
  • PDO PHP Extension
  • Composer Dependency Manager

You can get the latest version of PHP from php.net
also you can get Composer the PHP Dependency Manager from getcomposer.org
and you may need Git the distributed version control system from git-scm.com

Note: Make sure that the path of vendor/bin of your composer already added to your Environment Variables System, for example in Linux you must add this line to your system:
export PATH="$PATH:$HOME/.config/composer/vendor/bin"
