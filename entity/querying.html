Querying Data

The EntitySet property of EntityContext represents a collection of all entities of particular entity type, and can be used to query a single entity or multiple entities.

Loading All Entities

To load all the entities into an array, you can use the method toArray() of EntitySet property, as shown in the following example: 


                  <?php

                  namespace Application\Controllers;

                  use Artister\Web\Mvc\Controller;
                  use Artister\Web\Mvc\IActionResult;
                  use Artister\Web\Mvc\EntityManager;

                  class HelloController extends Controller
                  {
                     private EntityManager $DbManager;

                     public function __construct(EntityManager $dbManager)
                     {
                        $this->DbManager = $dbManager;
                     }

                     public function all() : IActionResult
                     {
                        $students = $this->DbManager->Students->toArray();
                        $listNames = "";

                        foreach ($students as $student) {
                           $listNames .= $student->Name .PHP_EOL;
                        }

                        return $this->content($listNames);
                     }
                  }

Note: The EntitySet is an iterable collection, so you can also iterate EntitySet instance using foreach statment, without using toArray() method, but it's not recommended to use it in the view template.

Loading a single entity

To load a single particular entity, use the method EntitySet::find(int $id) of EntitySet property to select an entity by its Id, like the following example: 


                  public function profile($id) : IActionResult
                  {
                     $students = $this->DbManager->Students;
                     $student = $students->find($id);
                     
                     $profile = "name: {$student->Name}\n";
                     $profile .= "Age: {$student->Age}";

                     return $this->content($profile);
                  }

Note: otherwise you can use the method EntitySet::first() to load to first entity of the record, or use the EntitySet::last() method to load the last one.

Linq Query

EntitySet implements IQueryable interface so it can use LINQ (Language-Integrated Query) to query data from the database, which allows you to use programing language to write strongly typed queries, by passing a representation of the LINQ query to the database provider in form of lambda expressions (closure function, arrow function) which can be muti functions chained together, and Database provider in turn translate it to database-specific query language (sqlite, mysql, ...) and return a data.

Linq methods are extension methods so in order to use this feature you have to define the extension namespace Artister\System\Linq; using the key word use in every class, or code use Linq methods on instance of EntitySet.

Note: Queries are always executed against the database even if the entities returned in the result already exist in the tracker.

Filtering Data

Use the extension method EntitySet::Where(closure $predecate) to filter the data.
The following code is equivalent to "SELECT * FROM Student WHERE Name = 'Alice' AND Age > 16" 


                  <?php

                  namespace Application\Controllers;

                  use Artister\Web\Mvc\Controller;
                  use Artister\Web\Mvc\IActionResult;
                  use Artister\Web\Mvc\EntityManager;
                  use Artister\System\Linq;

                  class HelloController extends Controller
                  {
                     private EntityManager $DbManager;

                     public function __construct(EntityManager $dbManager)
                     {
                        $this->DbManager = $dbManager;
                     }

                     public function filter() : IActionResult
                     {
                        $student = $this->DbManager->Students
                                                   ->where(fn($s) => $s->Name == "Alice" && $s->Age > 16)
                                                   ->first();
                     }
                  }
Ordering Data

The following code is equivalent to "SELECT * FROM Student ORDER BY Name, Age" 


                  public function order() : IActionResult
                  {
                     $students = $this->DbManager->Students
                                                 ->orderBy(fn($s) => $s->Name)
                                                 ->thenBy((fn($s) => $s->Age)
                                                 ->toArray();
                  }

Note: For the reverse ordoring you can use the methods orderByDescending and thenByDescending

Complex Query

You can use filer query where and ordering query like orderBy together.
The following code is equivalent to "SELECT * FROM Student WHERE Age > 16 ORDER BY Name" 


                  public function index() : IActionResult
                  {
                     $student = $this->Entity->Students
                                             ->where(fn($s) => $s->Age > 16)
                                             ->orderBy(fn($s) => $s->Name)
                                             ->toArray();
                  }
