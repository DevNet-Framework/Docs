Get Started

To create a MVC Web Application, Open the terminal then enter the following command: 


devnet new mvc --project YourProjectName

Or use composer: 


composer create-project devnet/mvc YourProjectName

Project structure

This command will create a mvc web project with the following structure: 


                  mvc-project/
                  ├─ Controllers/
                  │  ├─ HomeController.php
                  │  └─ AccountController.php
                  ├─ Models/
                  │  ├─ LoginForm.php
                  │  ├─ RegisterForm.php
                  │  └─ User.php
                  ├─ Views/
                  │  ├─ account/
                  │  │  ├─ index.phtml
                  │  │  ├─ login.phtml
                  │  │  └─ register.phtml
                  │  ├─ home/
                  │  │  └─ index.phtml
                  │  └─ Shared/
                  │     ├─ layout.phtml
                  │     └─ navbar.phtml
                  ├─ webroot/
                  │  ├─ css/
                  │  ├─ js/
                  │  ├─ lib/
                  │  ├─ .htaccess
                  │  ├─ index.php
                  │  └─ Web.config
                  ├─ ...
                  ├─ Program.php
                  ├─ project.phproj
                  ├─ Startup.php
                  └─ settings.json

MVC Registraction

Befor using MVC components, make sure the service MVC is registered inside the configureServices method of Startup class, by calling the extension method addMvc() of the instance of the type IServiceCollection. 


                  <?php
                  
                  namespace Application;
                  
                  use DevNet\Core\Dependency\IServiceCollection;
                  use DevNet\Core\Dispatcher\IApplicationBuilder;
                  use DevNet\Core\Extensions\DependencyExtensions;
                  use DevNet\Core\Extensions\HostingExtensions;
                  
                  class Startup
                  {
                     public function configureServices(IServiceCollection $services)
                     {
                        $services->addMvc();
                     }
                      ...
                  }

Conventional Routing

The default routing of MVC application is Conventional Routing, only one generic route template : {controller=Home}/{action=Index}/{id?} which can handle all url requests, by parsing it into 3 segments:

  1. Controller name: default name is "home"
  2. Action name: default name is "index"
  3. Id parameter: is optionnal

In order to use the Conventional Routing, you have to make sure that the Middlewars RouterMiddlewar and EndpointMiddlewar are added to the pipeline in the method configure(IApplicationBuilder $app) of the Startup class, with the help of the Extension methods addRouter() and addEndpoint(RouteCollection $routes). 


                  class Startup
                  {
                     ...                     
                     public function configure(IApplicationBuilder $app)
                     {
                        $app->useRouter();

                        // Middlewars here
                        
                        $app->useEndpoint(function($routes)
                        {
                           $routes->mapRoute("default", "{controller=Home}/{action=Index}/{id?}");
                        });
                     }
                  }

Note: if you add a Literal route with Conventional route make sure to put the conventional route at the buttom of the routes list, because it has a greedy pattern and will catch all the urls.

Development Server

DevNet framework use PHP Built-in development server, as default server to run the DevNet web application, assuming ./webroot/ is public accessible directory with index.php file, you can start the development server by running the following command in the terminal: 


devnet run

This will start the development server at http://localhost:8000 


devnet run
[Sat Dec 12 15:10:02 2020] PHP 8.0.0 Development Server (http://localhost:8000) started

Otherwize you can also start the Built-in PHP server, by runing the following PHP command line in the terminal: 


php -S localhost:8000 -t webroot
